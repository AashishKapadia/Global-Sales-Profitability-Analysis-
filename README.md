# Global-Sales-Profitability-Analysis-
Global Sales &amp; Profitability Analysis explores 100K records across 185+ countries (2010–2017) using Python, Tableau, and Kaggle data. It reveals $133.6B revenue, $39.4B profit, 29.5% margin, with Sub-Saharan Africa &amp; Europe leading, Cosmetics as top profit driver, and North America as untapped growth.
